transform

(transform configs)

Converts a nested dict(2nd level max) into a flat format with keywords as keys. Default key’s values are applied as they are Given { “default” { “a” 1 “b” 2} “email” { “host” “gmail.com” “port” 587} } transform into {:email-port 587, :a 1, :b 2, :email-host “gmail.com”} See that {default { a 1}} transformed => { :a 1 }
